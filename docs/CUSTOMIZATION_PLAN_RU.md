# План доработки под пользователя

## Выбранные параметры пользователя

- Название сборки: **akso alt**.
- Язык интерфейса: **полностью русский**.
- Опасные/расширенные функции: **оставить**, но помечать и документировать риски.

## Цель

Сделать свою удобную версию Roblox Account Manager, где:

- понятно, что делает каждая кнопка;
- интерфейс можно использовать без чтения исходников;
- опасные функции явно отмечены;
- настройки по умолчанию безопаснее;
- код проще поддерживать.

## Этап 5. Окно первого запуска

При первом запуске показывать wizard:

1. Выбор языка: `Русский / English`.
2. Выбор режима:
   - `Безопасный` — без webserver, без copy token, без save passwords;
   - `Обычный` — как сейчас, но с предупреждениями;
   - `Developer` — всё доступно, но только после подтверждения.
3. Выбор шифрования:
   - парольное — рекомендовано;
   - Windows DPAPI;
   - без шифрования — не показывать без advanced mode.
4. Краткое предупреждение про `.ROBLOSECURITY`.

## Этап 6. Рефакторинг

Минимальный безопасный рефакторинг:

1. Не переписывать весь UI сразу.

   - `SettingsService`;

   - `RiskWarningService`;

3. После этого трогать запуск Roblox и Web API.
