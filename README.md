# akso alt

`akso alt` — кастомная русская сборка проекта Roblox Account Manager.

## Что уже изменено

- Название окна приложения: `akso alt`.
- Метаданные сборки: `akso alt`.
- Добавлен runtime-перевод интерфейса на русский через `RussianLocalization`.
- Переведены основные окна:
  - главное окно;
  - настройки;
  - инструменты / список серверов;
  - утилиты аккаунта;
  - импорт;
  - редактор темы;
  - Account Control / Nexus;
  - часть пользовательских контролов.
- Опасные функции **не удалены**, но в русском интерфейсе помечены значком `⚠`:
  - копирование пароля;
  - копирование `username:password`;
  - копирование Security Token / `.ROBLOSECURITY`;
  - получение Authentication Ticket;
  - копирование rbx-player/app ссылок.
- Добавлен современный тёмный стиль по умолчанию.
- Добавлен встроенный справочник `Справочник akso alt`.
- Добавлены полезные пункты меню:
  - `Сделать backup AccountData`;
  - `Открыть папку программы`;
  - `Очистить буфер обмена`.

## Где лежит перевод

Файл:

```text
RBX Alt Manager/Classes/RussianLocalization.cs
```

Перевод применяется после `InitializeComponent()`, чтобы не править вручную `.Designer.cs`.
Так Visual Studio Designer не будет затирать русские подписи.

## Важное предупреждение

Функции, связанные с `.ROBLOSECURITY`, паролями, authentication ticket и web API, дают доступ к аккаунтам. Не отправляй эти данные другим людям.

## Документация

- `docs/FUNCTIONS_RU.md` — карта функций проекта на русском.
- `docs/CUSTOMIZATION_PLAN_RU.md` — план дальнейшей доработки.

## Что можно сделать дальше

1. Перевести сообщения `MessageBox` и тексты ошибок.
2. Добавить отдельное окно справки внутри программы.
3. Добавить подтверждения перед критическими действиями.
4. Сгруппировать интерфейс в режимы: `Обычный` / `Расширенный` / `Разработчик`.
5. Постепенно вынести большой `AccountManager.cs` в сервисы.
