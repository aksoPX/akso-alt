## 📖 О проекте

**akso alt** — изменённая русская версия менеджера аккаунтов Roblox с современным интерфейсом на **HTML / CSS / JavaScript** через **Microsoft WebView2**. Приложение сохраняет C#-логику оригинала, но интерфейс перенесён в WebUI.

> 🔗 Основа: [ic3w0lf22/Roblox-Account-Manager](https://github.com/ic3w0lf22/Roblox-Account-Manager) · Лицензия: GPL-3.0

---

## ✨ Возможности

- 🎨 Современный русский WebUI-интерфейс
- 🔁 Multi Roblox — несколько аккаунтов одновременно
- 🎯 Запуск по Place ID / Job ID / VIP-ссылке
- 💾 Сохранение игр, VIP-ссылок, Job ID
- 🌈 Цветовые темы, backup, проверка рисков

---

## 📥 Установка

1. Скачай релиз со страницы [**Releases**](../../releases)
2. Запусти `akso-alt-setup.exe` — WebView2 установится сам
3. Запускай через ярлык в меню Пуск

> ⚠️ Требуется Windows 10/11 и .NET Framework 4.7.2+

---

## 🔒 Безопасность

> [!WARNING]
> Никогда не публикуйте `AccountData.json`, `RAMSettings.ini`, `proxies.txt`, `Backups/` — они дают доступ к аккаунтам. Все они в `.gitignore`.
