## 🎮 akso alt v1.0.0

Русская WebUI-версия Roblox Account Manager.

### 📥 Установка
1. Скачайте **akso-alt-setup.exe** ниже
2. Запустите установщик — WebView2 установится сам
3. Запускайте через ярлык в меню Пуск

### ✨ Возможности
- Современный русский WebUI-интерфейс
- Multi Roblox, запуск нескольких аккаунтов
- Запуск по Place ID / Job ID / VIP-ссылке
- Темы, backup, проверка рисков

### 💻 Требования
- Windows 10/11, .NET Framework 4.7.2+, WebView2 (ставится автоматически)
